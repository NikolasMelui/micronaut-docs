6.5 Originating IP Addresses

Version: 1.2.0.BUILD-SNAPSHOT

6.5 Originating IP Addresses

You may need to resolve the originating IP address of an HTTP Request. Micronaut ships with several beans (HttpHeaderRequestIpAddressesResolver, RemoteAddressRequestIpAddressesResolver) of type RequestIpAddressesResolver and a composite implementation CompositeRequestIpAddressResolver which combines all registered RequestIpAddressesResolver and it is annotated with Primary.

The properties used by HeaderIpAddressesResolver can be customized by overriding the values of:

🔗
Table 1. Configuration Properties for HttpHeaderRequestIpAddressesResolverConfigurationProperties
Property Type Description

micronaut.http.originating-ips.header.header-name

java.lang.String

The HTTP Header name. Default value ("X-Forwarded-For").

micronaut.http.originating-ips.header.delimiter

java.lang.String

The delimiter used to separate IPs in the Http Header value. Default value (",")

micronaut.http.originating-ips.header.enabled

boolean

Whether the HTTP Header IpAddresses Resolver is enabled. Default value (true).

You can disable RemoteAddressIpAddressesResolver by setting micronaut.http.originating-ips.remote-address.enabled to false.
